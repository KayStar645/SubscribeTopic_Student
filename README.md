# Subscribe Topic

## System required

* NodeJs
* Yarn
* GIT
* Dot NET SDK

## IDE

* Visual studio code

## Start front end project

```terminal
    yarn
    yarn dev
```

## Start back end project

```terminal
    cd ../your_be_project/UI.WebAPi
    dotnet run
```
## Build project

```terminal
    yarn build
```

## Start build

```terminal
    yarn start
```
